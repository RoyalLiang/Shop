# 关于本项目
1. 基于国际化的商品推荐网站
    1. 商品系统
    2. 公司系统
    3. 后台系统
2. 基于Python的Django框架
    1. Python3
    2. Django2
    3. pure-paginator
    4. import-export
3. 后台使用Django自带Admin，对其稍微进行了扩展
4. 无注册登录功能
5. 使用了celery进行定时与异步任务
    1. 发送邮件
    2. 数据统计
6. 基本功能已完善，后续观察运行情况
